# QUESTIONÁRIO

## Pergunta 1

Explique a diferença entre linguagens compiladas e linguagens interpretadas, dando exemplos de cada uma.\
R: A principal diferença é que enquanto a linguagem interpretada(como Python e JS) é traduzida para código de máquina de
forma sequencial linha após linha, a linguagem compilada(como C e C++) já faz essa tradução antes mesmo da execução,
portanto se torna mais ágil.

## Pergunta 2

Compare as vantagens e desvantagens de usar uma IDE em vez do terminal para executar scripts Python. Dê exemplos
específicos para justificar sua resposta.\
R: As vantagens em trabalhar com IDE durante o desenvolvimento dos seus sistemas são várias, como recursos integrados e
depuração enquanto alguns pontos de desvantagem seria justamente o uso excessivo de ferramentas a ponto de causar
dependência e também é claro, se trata de um software muito mais pesado.

## Pergunta 3

Explique os conceitos de compilador e interpretador para a linguagem de programação\
R: para compiladores e interpretadores podemos seguir a mesma logica analisada acima. O interpretador é responsável por
fazer a tradução do código para linguagem de máquina linha a linha, já o compilador faz essa tradução uma unica vez no
código inteiro.

## Pergunta 4

Cite as principais diferenças e funcionalidades que você observou com relação ao Jupyter Notebook e o Pycharm.\
R: Jupyter por ser principalmente executado pelo navegador se torna algo para códigos rápidos, análise de dados entre
outros projetos menores, ja o pycharm contra com muitas facilidades, extensões e recursos para projetos maiores, mais
complexos e robustos como controle de versão, por exemplo.
